Team 4 이데댕김
항목	내용
프로젝트명	블록체인 비전문가 의사결정자를 위한, Fine-tuned CodeBERT 기반 스마트 컨트랙트의 보안 취약점 분석 및 LLM 기반 솔루션 리포트 제공 자동화 웹 서비스
프로젝트 키워드	블록체인 기반 스마트 컨트랙트, 코드 내 보안 결함 탐지 및 분석, 생성형 AI 기반 취약점 분석 리포트 생성
무엇을 만들고자 하는가	스마트 컨트랙트의 보안 취약점을 자동 탐지하고, 해킹 시뮬레이션을 통해 보안 점수를 제공하며, 이해하기 쉬운 분석 리포트 제공 및 코드 수정 제안으로 개인 및 소규모 기업 내 블록체인 비전문가가 안전한 계약서를 합리적인 가격으로 배포할 수 있도록 지원하는 웹서비스 입니다.
고객	김이화 (33세, 스타트업 대표)
- 고객 경험 : 독서 애호가 커뮤니티 웹서비스를 운영하는 스타트업의 대표다. 직원은 10명 이내로 블록체인 전문가는 없다. 이번에 커뮤니티 토큰으로 아이콘, 배지 등을 구매할 수 있는 기능을 추가하려고 한다. 스마트 컨트랙트 구현은 블록체인 개발자 1명에게 외주를 주었지만, 배포 후 수정이 어렵다보니 배포하기 전에 보안 문제가 있는지 한 번 더 확인하고 싶다.
- 고객 요구:
1. 저렴한 가격 : 보안 감사 서비스를 알아보니 전문가가 레포트를 써주는 건 가격이 월 $200 정도로 비싸다. 추가하려는 게 메인 기능이 아니고, 이미 고정 지출이 많아서 금액이 부담스럽다.
2. 쉬운 설명 : 회사 내에 블록체인 전문가가 없기 때문에 비전문가의 수준에 맞게 설명해주었으면 좋겠다.

박길동 (54세, 마케터 직무 직장인)
- 고객 경험: 회사에 다니며 부업으로 부동산 투자 강의 컨텐츠를 판매 중이다. 요즘 트렌드인 블록체인을 자신도 써보고 싶다. 그래서 유료 멤버십과 프리미엄 컨텐츠 판매에 블록체인 기반 커뮤니티 토큰을 도입하려고 한다. 스마트 컨트랙트 구현은 지인의 대학생 딸에게 외주를 맡겼다. 그런데 경력이 없는 주니어 개발자가 작성한 코드라서 보안 문제가 없을지 확신하기 어렵다.
- 고객 요구:
1. 저렴한 가격 : 부업 수입이 그렇게 크지 않아서 이 일에 큰 돈을 쓰고 싶지는 않다.
2. 쉬운 설명 : 블록체인 공부도 해보았지만, IT업계 사람이 아니다 보니 기술적인 부분은 도통 모르겠다. 어떤 위험이 있다는 건지 어려운 용어 없이 쉬운 언어로 풀어서 알려주면 좋겠다.
Pain Point	1. 스마트 컨트랙트는 한 번 배포하면 수정이 불가능하므로 사전에 보안 점검을 하지 않으면 배포 이후 취약점을 수정할 수 없으며, 이로 인해 금전적인 손실이 발생할 수 있다.
2. 블록체인은 보안 취약점이 지속적으로 지적되는 분야이기에, 실제 해킹을 당하면 대규모 데이터 유출과 자산 손실이 발생할 수 있다.
3. 기존의 스마트 컨트랙트 보안 감사 외주를 받는 개인 컨설턴트나 전문 감사 업체는 개인과 소규모 기업이 사용하기에 비싼 비용 부담이 있다.
4. 기존 서비스들은 CLI 기반으로 서비스가 제공되어, 비전문가들이 사용하기에 기술적인 장벽이 존재한다.
기술 솔루션 서술	1. 스마트 컨트랙트의 보안 취약점 중 대표적인 Reentrancy Attack과 Integer Overflow/Underflow를 탐지하고 대응 방안(코드 수정)을 제시하는 기능을 우선적으로 제공하는 솔루션
2. 스마트 컨트랙트 보안 취약점 데이터를 기반으로 Slither API와 CodeBERT AI모델을 활용하여, 코드 내 보안 결함을 자동으로 탐지 및 분석하여 자동화하는 솔루션
3. 해킹 시뮬레이션 오픈 소스 (echidna)를 통해 보안 점수 및 개선 방안을 제공하는 솔루션
4. 보안 분야의 비전문가도 쉽게 이해할 수 있도록, 생성형 AI(GPT-4.5)를 활용해 쉬운 용어로 분석 리포트 생성을 자동화하는 솔루션
구현기술 키워드	블록체인 보안, Slither, CodeBERT, echidna, GPT-4.5
사용할 소프트웨어의 구체적 명세(명칭과 URL)	스마트 컨트랙트 보안 분석 도구

- 이더리움 해킹 시뮬레이션 오픈소스 : echidna (https://github.com/crytic/echidna)

    <td><strong>무엇을 만들고자 하는가</strong></td>
    <td>
      보호자는 치료 자체보다 정보를 몰라 생기는 행정·재정 부담에 큰 스트레스를 받는다. 특히 산정특례, 지원제도, 급여 기준, 보험 청구 가능성은 치료 단계가 바뀔 때마다 계속 변화한다. CareMap은 초진 → 수술 → 항암 → 만성 관리 등 치료 여정 전체를 따라가며, 단계가 바뀔 때마다 필요한 정보를 반복적으로 안내하는 서비스이다. 사용자는 질병·치료 단계를 선택하기만 하면 현재 시점에서 해야 하는 행정 절차를 자동으로 확인할 수 있다. 또한 일정 관리와 산정특례 만료일도 자동으로 안내한다.
    </td>
  </tr>

  <tr>
    <td><strong>고객(페르소나)</strong></td>
    <td>
      ① 신부전/루푸스 환자 보호자 – 산정특례·지원제도·약값 기준 등을 반복해서 확인해야 함<br>
      ② 희귀암 환자의 자녀 – 항암제 변경, 재발 여부 등 치료 단계에 따라 행정적인 정보가 매번 달라지며 지속 확인 필요<br>
      ③ 만성질환 부모를 돌보는 직장인 – 주기적인 검사·입원·치료 변화에 따라 반복적으로 행정 체크가 필요함
    </td>
  </tr>

  <tr>
    <td><strong>Pain Point</strong></td>
    <td>
      1) 질병별 제도 혜택을 스스로 찾기 어려움<br>
      2) 제도별 조건·서류·소득 기준이 서로 달라 비교가 어려움<br>
      3) 보험 청구 가능 항목을 직접 약관에서 찾기 어려움<br>
      4) 치료 단계가 바뀔 때마다 필요한 절차가 달라져 보호자는 주기적으로 다시 찾아봐야 하는데 이를 관리해주는 시스템이 없음
    </td>
  </tr>

  <tr>
  <td><strong>핵심 기능 (상세 예시 포함)</strong></td>
  <td>
<strong>① 질병 카테고리 기반 제도 추천</strong><br>
- '희귀암', '신부전', '자가면역질환' 등 질병 버튼 선택 → 해당 질환의 산정특례 가능 여부, 본인부담률 변화, 신청 절차 자동 요약<br>
예시: <em>C49 희귀암 → 산정특례 등록 가능성 높음 / 본인부담률 95%→5% / 필요한 서류 요약 제공</em><br><br>

<strong>② 치료 단계 기반 “액션 플랜” (반복 사용의 핵심 구조)</strong><br>
- 초진/수술 후/항암 시작/재발 등 단계 선택 → 해당 단계에서 “지금 해야 할 일 3가지” 제시<br>
- 치료 단계가 바뀔 때마다 자동으로 필요한 행동 리스트 갱신 → 반복 이용 구조<br>
예시: <em>항암 시작 전: 산정특례 등록 확인 / 항암제 급여 기준 확인 / 입원 영수증 준비</em><br><br>

<strong>③ 최소 입력 보험 체크 (키워드 안내 중심)</strong><br>
- 보장 유형 체크박스만 선택하면 간단한 후보 안내 (암 진단비, 고액암, 입원일당 등)<br>
- 금액 판단 X / 청구 가능성 높은 항목 + 약관 키워드 안내<br>
예시: <em>암 진단비 선택 → “최초 진단일 청구 가능성 있음 / 약관에서 '고액암' 키워드 확인”</em><br><br>

<strong>④ 일정·만료 알림 기능 (지속적 사용 유도 핵심)</strong><br>
- 진료·검사 일정 등록<br>
- 산정특례 만료 3개월/1개월 전 자동 알림 제공<br>
- 지자체 지원사업 공고 알림<br><br>

<strong>⑤ 제도 설명 AI 요약</strong><br>
- 여러 제도를 읽어야 하는 부담을 줄이기 위해 긴 문서를 2~3줄로 요약해 제공
  </td>

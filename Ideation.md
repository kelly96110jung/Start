# Ideation - 의료 언어 자동 해석·요약 앱 (Version 1.0)

---

**Key words**  
Healthcare · STT · AI · Medical-Term Simplification · Patient-Education · On-device Processing

---

## 누구를 위해

의학 용어를 이해하기 어려운 환자·보호자,  
진료실에서 들은 설명을 기억하기 힘들어하는 고연령층,  
의료 정보를 한 번에 정리하기 어려운 사람들

---

## 누가 어떤 문제 때문에 사용하는지

- 진료실에서는 의사 설명을 빠르게 듣느라 핵심 내용을 기억하지 못함  
- ‘크레아티닌’, ‘전해질 불균형’, ‘삼차신경통’ 등 전문 용어가 어려움  
- 외래 진료 시간이 짧아 충분히 질문하거나 내용을 정리하기 어려움  
- 고연령층·청력 저하 환자들은 전문용어 중심의 설명을 이해하기 힘듦  
- 보호자가 함께 가지 못하면 집에 돌아가서 설명을 제대로 전달하지 못함  

---

## 어떤 기술을 사용할지

- **On-device STT(Android/iOS)**: 기기 내 음성 인식 기반 의료 설명 실시간 텍스트 변환  
- **GPT 기반 자연어 요약/재구성**: 전문 의료 용어를 환자 쉬운말로 변환  
- **Medical-term Dictionary(DB)**: 검사명·약물명·질환명·수치의 기본 정의 저장  
- **FlutterFlow 또는 React Native**: 크로스플랫폼 앱 개발  
- **Firebase + Functions**: 사용자 프로필·진료 기록 관리(민감 정보 최소 저장)

---

## 무엇을 만들고자 하는가

1. **진료 음성 → 실시간 STT 변환**  
   - 의사의 설명을 스마트폰에 녹음하면, 서버 전송 없이 기기 내에서 텍스트로 변환  
   - 민감 정보 보호, 오프라인 환경에서도 동작 가능  

2. **의료 용어 자동 감지 및 쉬운말 해석**  
   - 변환된 텍스트에서 전문 용어를 자동 탐지  
   - 일반인 기준으로 이해하기 쉬운 설명으로 재작성  
   - 검사 수치·약물·질환명 등 태깅 제공  

3. **환자 친화적 요약 기능**  
   - 긴 설명 중 핵심 내용만 정리해 ‘오늘 진료 핵심 3가지’ 형태로 제공  
   - 예: “수치가 왜 높은지”, “다음 진료까지 주의할 점”, “약 복용 방법”  

4. **진료 기록 보관 + 비교 기능**  
   - ‘오늘 진료 기록’으로 저장  
   - 같은 질환/검사 항목의 변화 추적  
   - 이전 진료와 비교하여 쉽게 이해 가능
